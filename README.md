# GATR2 Push-Back worlds


GATR2 is one of the University of Florida’s VEX U robotics teams, part of the Gator VEX Design Team.  
This repository contains the codebase used for development, testing, and competition.

## Overview

GATR2 focuses on:
- Competitive VEX U robotics development
- Clean, modular code for rapid iteration
- Reliable autonomous and driver control systems
